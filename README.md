# Radio Drift

A minimalist web app for ambient audio mixing.

**[Live Demo](https://jgresula.github.io/radio-drift/)**

## Features

**Radio Roulette** — Discover internet radio stations by genre. Shuffle through thousands of streams worldwide.

**Ambient Soundscape** — Mix curated ambient radio with live air traffic control for focus and relaxation.

## Tech

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies.

- Radio stations via [Radio Browser API](https://www.radio-browser.info/)
- Live ATC via YouTube embeds
